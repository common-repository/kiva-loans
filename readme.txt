Shows your latest Kiva Loans in a widget.

== Description ==

A simple plugin that displays your latest Kiva Loans in a widget. It shows the photo from the loan and the name 
and links this to relevant loan on Kiva. You can:

* Select how many loans you want to show in your widget.
* Show or Hide a Link to all your Kiva Loans on your Kiva Loan page.
* Show or Hide a Link to further information about Kiva on Kiva's invite page.

You can see this in action on http://www.cultivateddrivel.com/


== Installation ==

1. Upload 'kiva-loans.php' to the '/wp-content/plugins/' directory
2. Upload 'widgets/kiva-loans.php' to the '/wp-content/plugins/widgets/' directory
3. Activate the pugin through the 'Plugins' menu in WordPress
4. Enable the widget on your sidebar in the Appearance->Widgets menu in Wordpress
